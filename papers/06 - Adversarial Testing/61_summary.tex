\section{Summary}
\draft{%
    \begin{itemize}
        \item \gls{dnn} driven vehicles heavily dependent on manual collection of data under different driving conditions
            \begin{itemize}
                \item labeled data
                \item \(=>\) very expensive
                \item unguided simulations
                \item \(=>\) miss numerous corner cases
            \end{itemize}
        \item Design, implement and evaluate DeepTest
        \item Systematic testing tool for automatically detecting erroneous behaviors of \gls{dnn}-driven vehicles that potentially lead to fatal crashes
        \item tool generate test cases leveraging real world changes in driving conditions
        \item systematically explore different parts of the \gls{dnn} logic by generating test inputs that maximize the number of activated neurons
        \item \(=>\) Found thousands of erroneous behaviors under different realistic driving conditions
        \item neuron coverage = good approximation for estimating diversity of output
        \item neuron coverage increases adding transformed images and combinations of transformations
    \end{itemize}
}
